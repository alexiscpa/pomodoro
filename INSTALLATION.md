# Quick Installation Guide

## Step 1: Generate Icons (Required)

The extension needs icon files to work. Choose one of these options:

### Option A: Auto-generate with Python (Recommended)
```bash
cd icons
python generate_icons.py
```

### Option B: Manual Creation
Create three PNG files in the `icons/` directory:
- `icon16.png` (16Ã—16 pixels)
- `icon48.png` (48Ã—48 pixels)
- `icon128.png` (128Ã—128 pixels)

Use any image editor or online tool. A simple red circle works fine for testing.

## Step 2: Load Extension in Chrome

1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right)
4. Click **Load unpacked**
5. Select the `pomodoro` folder
6. Done! The extension is now installed

## Step 3: Start Using

1. Click the FocusFlow icon in your Chrome toolbar
2. Click **Start** to begin your first Pomodoro session
3. Stay focused for 25 minutes!
